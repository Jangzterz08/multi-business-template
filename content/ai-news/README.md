# AI News Records

Generated digests land here.

- `latest.json`: machine-readable snapshot of the newest run
- `YYYY-MM-DD.md`: daily markdown digest of the top five AI stories
- `YYYY-MM-DD-content-plan.md`: ready-to-create brief with hooks, subject lines, scripts, and verification steps
- `YYYY-MM-DD-daily-posting-brief.md`: one-file morning briefing with what to open and publish first
- `YYYY-MM-DD-instagram-carousel.md`: Instagram carousel outline, caption, and alternate cover options
- `YYYY-MM-DD-instagram-reel.md`: Instagram Reel script, caption pack, alternate hooks, and CTA options
- `YYYY-MM-DD-linkedin-carousel.md`: slide outline, caption, and alternate cover options for a LinkedIn carousel
- `YYYY-MM-DD-newsletter.md`: first-draft morning newsletter
- `YYYY-MM-DD-quality-report.md`: publishing QA scorecard with hook, CTA, and redundancy checks plus auto-fix options when a draft is weak
- `YYYY-MM-DD-publishing-queue.md`: the exact morning posting order
- `YYYY-MM-DD-publishing-checklist.md`: preflight and success-metric checklist
- `YYYY-MM-DD-talking-head-30s.md`: timed 30-second talking-head script with alternate hooks and CTA options
- `YYYY-MM-DD-video-scripts.md`: three short-form video scripts
- `YYYY-MM-DD-x-thread.md`: lead-story X thread draft
- `YYYY-MM-DD-social-posts.md`: five social post drafts

The GitHub Actions workflow at `.github/workflows/ai-news-content.yml` refreshes this folder on a schedule or when run manually.
