# Daily Operator Checklist

Use this every morning after the AI news workflow runs.

Need the phone version instead? Use [POST_THIS_NOW_CHECKLIST.md](./POST_THIS_NOW_CHECKLIST.md).

## 5-minute routine

1. Open `content/ai-news/YYYY-MM-DD-ready-to-post.md`
2. Open `content/ai-news/YYYY-MM-DD-daily-posting-brief.md`
3. Open `content/ai-news/YYYY-MM-DD-quality-report.md`
4. Verify the lead story on the original source if it is sensitive, surprising, or unclear
5. Fix any weak hook, repeated opener, or weak CTA using the built-in alternate hooks and CTA options
6. Decide whether to keep the lead story or switch to a stronger backup story

## Publish order

1. Publish the newsletter
2. Publish the short-form video
3. Publish the LinkedIn or Instagram carousel
4. Publish a backup story later only if the topic is still hot

## Before you hit publish

- Add one original takeaway
- Keep attribution clear
- Make sure the first line is strong
- Make sure the close asks for one clear action

## Track after posting

- Newsletter: clicks and replies
- Video: 3-second hold rate and completion
- Instagram: saves, shares, comments
- LinkedIn: comments, saves, reposts
- X: replies, reposts, bookmarks
